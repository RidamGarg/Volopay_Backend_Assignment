## API endpoints

- `/api/total_items`
- `/api/nth_most_total_item`
- `/api/percentage_of_department_wise_sold_items`
- `/api/get_monthly_sale`

### Steps

1. Install NPM packages
   ```sh
   npm i
   ```
2. Create a ``.env` file, and add the mongodb url.
3. Seed the database

```sh
node importData.js
```

4. Run the server

```sh
node index.js
```
